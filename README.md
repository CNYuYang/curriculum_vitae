## 实习经历模板

```tex
% Experience Template
\item[] {\RoleFont Experience Role} \\
\normalfont Company \\ 
{\DateFont StartDate - EndDate \hfill Location } \\ 
{\color{CV_Color}\LARGE - }{Description}
```
